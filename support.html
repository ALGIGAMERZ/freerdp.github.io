Support

Getting Help

If you need help the easiest way is to get in touch with us is our Mailing List or our IRC channel #freerdp @ on irc.freenode.net.

Bugs and feature requests

Bugs, feature requests and possible enhancements can be reported at our GitHub page. Have a look to BugReporting for details.

Documentation

Documentation is spread across multiple places. The most comprehensive (but still incomplete) documentation can be found in our Wiki. For developers additionally the documents in the "docs" of the source tree as well as the daily updated API documentation might be of interest.
